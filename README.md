# StereoGen documentation

This repository contains public StereoGen user documentation, privacy
information, screenshots, and test images.

- [English user guide](user-guide.en-US.md)
- [简体中文用户指南](user-guide.zh-CN.md)
- [带轮廓的三维立体图简易教程](tutorial-outline.zh-CN.md)
- [带皮肤的三维立体图简易教程](tutorial-skinned.zh-CN.md)
- [Privacy policy](privacy-policy.md)
- [Test images](test-assets/)

The published website is expected at
[giantpand2000.github.io/stereogen-docs](https://giantpand2000.github.io/stereogen-docs/).

The Markdown files are the source of the HTML website built by GitHub Pages.
`_config.yml` contains the Jekyll configuration.

## Repository contents

- `index.md` — website landing page
- `user-guide.en-US.md` — English user guide
- `user-guide.zh-CN.md` — Simplified Chinese user guide
- `tutorial-outline.zh-CN.md` — outline-stereogram tutorial in Simplified Chinese
- `tutorial-skinned.zh-CN.md` — skinned-stereogram tutorial in Simplified Chinese
- `tutorials/outline/` — tutorial script, source images, results, and screenshots
- `tutorials/skinned/` — source images, results, and screenshots for the skinned-stereogram tutorial
- `privacy-policy.md` — public privacy statement
- `store-listing/` — customer-visible Microsoft Store screenshots
- `test-assets/` — sample images for learning and certification testing

Keep operational submission records and all account, tax, payout, signing, and
credential information outside this directory.
