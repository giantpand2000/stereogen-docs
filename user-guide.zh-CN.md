# StereoGen 用户指南

StereoGen 是一个从深度图和纹理生成单图立体画（autostereogram）的桌面应用。
生成过程完全在本机完成，不需要账号，也不会把输入图片或结果上传到开发者的服务器。

![使用现代算法生成的单图立体画]({{ site.baseurl }}/store-listing/zh-CN/screenshots/01-modern-stereogram-light.png)

*使用“现代”算法生成的单图立体画。*

## 系统要求与安装

- Windows 10 或 Windows 11。
- 建议[通过 Microsoft Store 安装](https://apps.microsoft.com/detail/9N7Q261JQW8C)，以便获得签名和后续更新。
- 应用界面使用系统 WebView2。Windows 11 通常已经包含该组件；如果系统提示安装或更新，请使用微软提供的运行时。

安装完成后，从“开始”菜单打开 **StereoGen**。

## 快速开始

1. 打开“深度图”标签页，点击 **打开深度图…**。
2. 打开“纹理”标签页，点击 **打开纹理…**。
3. 切换到“立体图”标签页。
4. 保持默认的“现代”算法，点击 **生成**。
5. 使用 **适应**、**100%**、加号和减号检查结果。
6. 点击 **另存为…**，导出 PNG 或 JPEG。

也可以把图片拖入应用：拖入一张图片时，它会进入当前标签页（默认是深度图）；
同时拖入两张图片时，第一张作为深度图，第二张作为纹理。

想进一步制作轮廓清晰、图层可调的作品，可以继续阅读
[带轮廓的三维立体图简易教程]({{ site.baseurl }}/tutorials/outline-stereogram/)。

## 准备输入图片

### 深度图

深度图用亮度或所选颜色通道表达距离。通常黑色表示较远区域，白色表示较近区域，
灰色表示中间深度。为了获得稳定结果，建议使用边缘清晰、灰度过渡平滑的图片。

### 纹理

纹理是隐藏深度结构时重复使用的图案。细节丰富、局部对比适中的无缝纹理通常更容易观看。
过大的纯色区域或非常强烈的边缘可能让重复结构更加明显。

### 支持的格式

输入支持 PNG、JPEG、BMP、GIF 和 TGA。TIFF 当前不受支持。输出可保存为 PNG 或 JPEG。
需要保留透明背景时，请保存为 PNG。

可以使用文档仓库中的 [`depth-map.png`]({{ site.baseurl }}/test-assets/depth-map.png)
和 [`texture.png`]({{ site.baseurl }}/test-assets/texture.png) 完成第一次测试。

## 渲染算法

| 算法 | 适用场景 |
|---|---|
| 现代 | 推荐的通用模式；适合大多数深度图和纹理。 |
| 经典 · 左到右 | 按从左到右的经典规则生成。 |
| 经典 · 右到左 | 按从右到左的经典规则生成；可用于比较不同传播方向。 |
| 映射纹理 | 使用与深度图逐像素对应的纹理/材质图；两张输入图必须尺寸相同。 |

如果不确定，从“现代”开始。更改算法或其他选项后，重新点击 **生成**；也可以启用
**自动生成**，让应用在选项变化后自动刷新结果。

## 观看方式

### 发散（平行眼）

把焦点放到画面后方，像看远处一样放松双眼。结果上方的两个引导点看起来合并为三个点时，
保持眼睛状态，再慢慢把注意力移到下方图像。

### 会聚（交叉眼）

把焦点放到屏幕前方，使双眼轻微交叉。两个引导点合并为三个后，再观察下方图像。

如果深度方向与预期相反，可在“发散”和“会聚”之间切换。初次练习时，降低视差、
让窗口距离眼睛稍远，并避免长时间勉强聚焦。出现眼部不适时应立即休息。

## 常规选项

- **视差**：控制重复间距和立体效果强度。数值过大时可能难以融合，建议逐步增加。
- **拉伸 / 平铺**：决定纹理如何覆盖输出区域。
- **使用纹理尺寸**：以纹理宽度作为重复尺寸，并停用手动视差控制。
- **深度系数**：控制深度变化幅度；较低数值更容易观看，较高数值效果更强。
- **深度映射**：选择“线性曲线”可保持原版 StereoGen 的映射；“透视凸曲线”会让远处变化较慢、近处变化较密集。
- **曲线 k**：只在透视凸曲线中使用。`k` 越小，曲线越凸；`k` 越大越接近线性。
- **纹理起点**：现代算法中纹理进入重复周期的位置。
- **插值**：平滑图像采样，通常保持启用。
- **去除回声伪影**：减少重复轮廓造成的鬼影。
- **去除噪点**：清理生成结果中的孤立噪点。
- **深度通道**：从亮度、红、绿、蓝或透明通道读取深度。普通灰度图可使用“亮度”。

点击 **恢复默认** 可重置渲染选项。选项会保存在本机应用配置目录中，供下次启动使用。

### 视差、深度系数与 S

StereoGen 内部用 `S` 表示从最远端到最近端的**最大像素位移量**，也就是
`far separation - near separation`。它不是视差 `P` 本身，而是由视差和深度系数共同得到：

```text
P = 视差（像素），也是 far separation
f = 界面显示的深度系数百分数，例如 35
F = f / 100，例如 35% → 0.35

S               = (P - 1) × F
far separation  = P
near separation = P - S
```

启用“使用纹理尺寸”时，公式里的 `P` 取纹理宽度，而不是已停用的手动视差值。

例如，视差为 `100 px`、深度系数为 `35%` 时：

```text
S               = (100 - 1) × 0.35 = 34.65 px
far separation  = 100 px
near separation = 100 - 34.65 = 65.35 px
```

`P - 1` 中的 `1` 是原版 StereoGen 公式的一部分；因此即使深度系数为 100%，
最近端间距仍保留为 `1 px`，不会变成零。计算过程允许小数像素，生成时再由采样和插值处理。

若深度图归一化值为 `t`（黑色最远为 `0`，白色最近为 `1`），线性曲线使用：

```text
当前位移 s(t)       = t × S
当前 separation D(t) = P - s(t) = P - t × S
```

透视凸曲线仍使用同一个 `S`，所以最远端仍为 `P`、最近端仍为 `P - S`；
`k` 只重新分配两端之间的中间深度。也就是说，改变 `k` 不会改变最大深度幅度，
只会改变灰度中间值映射到多少像素位移。发散/会聚模式会改变扫描时位移的方向，
不会改变 `S` 的大小。

## 映射纹理模式

“映射纹理”适合使用材质图或带透明区域的输入。深度图与纹理必须具有完全相同的像素尺寸。

- **背景色**：指定未被纹理覆盖区域的颜色。
- **无颜色**：使用透明背景；此时应导出 PNG。
- **容差**：决定哪些接近背景色的像素被视为背景。
- **清除背景**：按背景色和容差移除匹配区域。
- **自动填满画布**：自动重复散射/收集，直到画布不再获得新像素或达到安全上限。
- **重复次数**：未启用自动填满时，手动控制处理轮数。

## 预览和导出

- **− / +**：缩小或放大预览。
- **100%**：按实际像素显示。
- **适应**：让整张图片适应当前窗口。
- **另存为…**：保存最近一次成功生成的结果。

缩放只影响屏幕预览，不改变导出图片的分辨率。

## 常见问题

### “尚未加载深度图”或“尚未加载纹理”

分别打开两张输入图片后再生成。顶部标签页可以确认当前查看的是哪张图片。

### 映射纹理提示尺寸不同

将深度图和纹理裁剪或缩放为完全相同的宽度和高度，然后重新打开它们。

### 看不到立体效果

先使用默认“现代 + 发散”设置，降低视差和深度系数，并用文档中的测试素材练习。
确认两个引导点能稳定地看成三个点，再把注意力移到图像上。

### 结果太吵或出现重复轮廓

启用“插值”“去除回声伪影”和“去除噪点”，降低深度系数，或换用细节分布更均匀的纹理。

### 透明背景保存后变成实色

使用映射纹理的“无颜色”选项，并保存为 PNG；JPEG 不支持透明通道。

## 隐私

StereoGen 在本机读取用户主动选择的图片，并把结果保存到用户选择的位置。详情参见
[隐私政策]({{ site.baseurl }}/privacy-policy/)。
